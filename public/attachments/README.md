# Attachments Directory

Place files you want to attach to blog posts in this directory.

## Usage

Reference files in your blog post frontmatter like this:

```yaml
attachments:
  - name: "my-file.pdf"
    url: "/attachments/my-file.pdf"
    size: "245 KB"
    type: "pdf"
```

## File Types Supported

- PDF documents
- Text files
- ZIP archives
- Documents (DOC, DOCX)
- Spreadsheets (XLS, XLSX)
- And more!

## Best Practices

- Keep file sizes reasonable
- Use descriptive file names
- Include file size in the attachment metadata
- Specify the file type for better icon display
